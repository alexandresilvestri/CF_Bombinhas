# EazyBox

**This is a Check-in App for Crossfit Gyms**

## Stack

- Ruby On Rails
- React (Typescript)
- Tailwind
- PostgreSQL
- React Native

## Host (Full AWS)

- ECS + Fargate (API)
- S3 + CloudFront (Web Client)
- Aurora RDS (SQL Database)
- Cognito + SNS (Mobile)

### Getting started

Clone the project:
- Git: git clone https://github.com/alexandresilvestri/eazybox
- Github CLI: gh repo clone alexandresilvestr/eazybox

Run:
- run: docker compose up

Acess https://localhost:3000
